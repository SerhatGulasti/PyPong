# PyPong
Basic pong game written with Python (Turtle library). Needs Python 3 and Tkinter.

#### How to play
**Left Paddle**


**w:** Up


**s:** Down

If caps lock on, it doesnt work.


**Right Paddle**

**Up Arrow Key:** Up

**Down Arrow Key:** Down

**r:** Resets the game. I know it just resets ball positions I'll make an update for score reset.
